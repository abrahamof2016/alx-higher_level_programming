# 0x0A. Python - Inheritance

*INTRODUCTION*

Inheritance is a fundamental concept in object-oriented programming(OOP) that allows you to create new classes (subclasses) that inherit properties and behaviors from existing classes (parent classes).

This repository contains tasks related to python - Inheritance.

*PROJECTS*
